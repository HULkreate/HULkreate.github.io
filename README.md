# HULkreate — Beauty Influencer Program Website

A locally hosted static website for the HULkreate beauty influencer program.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)

## Setup

1. Open a terminal in this folder.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the local server:
   ```
   npm start
   ```
4. The website will open automatically at **http://localhost:8080**

## Pages

| Page | File | Description |
|------|------|-------------|
| Landing Page | `index.html` | Homepage with program info, benefits, and FAQs |
| Creator Application | `apply.html` | Application form for influencers |
| Confirmation | `received.html` | Thank-you page after form submission |
